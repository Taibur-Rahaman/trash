# Personal Notes & Drafts

A private scratch repository for coursework drafts, assignment notes, proposals, and work-in-progress documents.

## Overview

This repo is **not** a deployable application. It collects markdown files, text drafts, and miscellaneous academic/project notes used during development and writing.

## Contents

Typical files include:

- Course assignment drafts (`373.md`, `499.md`, `econometrics-assignment/`)
- Project proposals and reports (`proposal.md`, `reportarifin.md`)
- Meeting and script notes (`script.md`, `new.md`, `new1.md`, etc.)

## Usage

There is nothing to install or run. Browse the files directly on GitHub or clone locally for reference:

```bash
git clone https://github.com/Taibur-Rahaman/trash.git
```

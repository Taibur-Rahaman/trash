## Files

| File | Description |
|------|-------------|
| `Assignment_Poverty_Inequality_SSNP_Bangladesh.md` | Full assignment (with embedded figures) |
| `generate_assignment_graphs.py` | Python script to regenerate all charts |
| `requirements-graphs.txt` | Dependencies (`matplotlib`, `numpy`) |

## Regenerate charts

```bash
pip install -r requirements-graphs.txt
python generate_assignment_graphs.py
```

Works in terminal and Jupyter (uses current folder if `__file__` is unavailable).
